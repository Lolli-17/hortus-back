Il tuo compito è analizzare un blocco di notizie di cronaca. Devi restituire un **singolo oggetto JSON** che contenga due chiavi principali: "analisi_generale" e "analisi_individuale".

1.  **"analisi_generale"** (Oggetto): Deve contenere la tua sintesi dell'intero blocco di notizie.
    * `sentiment_complessivo`: Un singolo sentiment (es. "Rabbia", "Tranquillità", "Felicità") che descrive il "clima" generale di tutte le notizie.
    * `media_score_negatività`: La media matematica approssimativa di tutti gli score di negatività (da 0 a 10).
    * `categoria_dominante`: La categoria tematica più frequente nel blocco di notizie.

2.  **"analisi_individuale"** (Array): Deve contenere un oggetto JSON per ogni singola notizia analizzata.
    * `sentiment`: Uno tra "Felicità", "Tranquillità", "Calma" e i loro opposti "Tristezza", "Paura", "Rabbia".
    * `score_negatività`: Un punteggio da 0 a 10.
    * `categoria`: La categoria tematica della notizia.
    * `id_notizia`: L'ID numerico della notizia.

Restituisci **solo e unicamente l'oggetto JSON completo** che contiene entrambe le chiavi.

---

{
  "analisi_generale": {
    "sentiment_complessivo": "Paura",
    "media_score_negatività": 6.5,
    "categoria_dominante": "Criminalità"

  "analisi_individuale": [
    {
      "id_notizia": 1,
      "sentiment": "Paura",
      "score_negatività": 9,
      "categoria": "Criminalità"
    },
    {

    },
    {
      "id_notizia": 3,
      "sentiment": "Tristezza",
      "score_negatività": 8,
      "categoria": "Società"
    }
  ]
}
