Il tuo compito è analizzare un blocco di notizie di cronaca. Per ogni notizia fornita, devi restituire un oggetto JSON con la tua analisi del sentiment.

L'analisi per ogni notizia deve includere:
1.  "sentiment": uno tra "Felicità", "Tranquillità", "Calma" e i loro opposti "Tristezza", "Paura", "Rabbia".
2.  "score_negatività": un punteggio da 0 (per nulla negativo) a 10 (estremamente negativo).
3.  "categoria": una categoria tematica (es. "Economia", "Criminalità", "Politica", "Ambiente", "Guerra", "Società").
4.  "id_notizia": l'ID numerico della notizia che stai analizzando.

Restituisci **solo e unicamente un array JSON** che contiene gli oggetti di analisi per ogni notizia.

---
ESEMPIO DI FORMATO DELLA RISPOSTA (JSON ARRAY):
[
  {
    "id_notizia": 1,
    "sentiment": "Paura",
    "score_negatività": 9,
    "categoria": "Criminalità"
  },
  {
    "id_notizia": 2,
    "sentiment": "Calma",
    "score_negatività": 2,
    "categoria": "Politica"
  },
  {
    "id_notizia": 3,
    "sentiment": "Felicità",
    "score_negatività": 1,
    "categoria": "Società"
  }
]
---
